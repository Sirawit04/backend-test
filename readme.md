Defined Conventions

Language: Python 3

Directory Structure: Each algorithm challenge is contained within its own dedicated directory (e.g., odd_int, permutations, smiley_faces).

Testing Tool: We use Python's built-in unittest tool to check if our code works correctly.

How to Run Tests: You can run the tests straight from your computer's terminal by typing a command like python test_smileys.py.


Challenges Included

Odd Int (odd_int.py): Finds the integer that appears an odd number of times.

Smiley Faces (smileys.py): Counts the total number of valid smiley faces.
